# Retail Inventory Intelligence System (RIIS)

## Business Problem

Indian retail chains often face:

1. Dead Stock
2. Stockouts
3. Expiry Losses
4. Poor Markdown Decisions

These issues lead to significant financial losses.

## Project Goal

Build a Retail Inventory Intelligence System that:

- Detects inventory imbalance
- Recommends inter-store transfers
- Suggests markdown schedules
- Predicts stockout risks
- Estimates financial impact

## Target Users

- Store Managers
- Regional Managers
- Procurement Teams

## Final Deliverables

1. PostgreSQL Data Warehouse
2. Python Analytics Engine
3. Power BI Dashboard
4. Excel Reports
5. Streamlit Demo
